上节课我们讲了用户信息展示，今天我们来讲修改用户信息功能。

需求

用户修改自己的信息，如头像，昵称，手机号等，我们以修改昵称为例。

功能流程

通过用户凭据 token 获取到用户 id，在数据库 user 表中找到用户记录，把前端传来的昵称更新到记录中。
